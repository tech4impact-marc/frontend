## Build and Run

```
NEXT_PUBLIC_MAPBOX_API_KEY=
NEXT_PUBLIC_IP_ADDRESS=
NEXT_PUBLIC_KAKAO_API_KEY=
NEXT_PUBLIC_JAVASCRIPT_KEY=
NEXT_PUBLIC_WEBURL=
```

환경 변수 설정 후 아래의 커맨드를 실행하면 앱을 실행할 수 있습니다.

```
npm i
npm run build
npm run start
```
